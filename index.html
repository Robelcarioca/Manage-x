EAU / Admin / System Design

System Architecture & Improvements

Database schema, suggested features, hierarchy design, and automation opportunities

🏗 Organizational Hierarchy & Workflow

UNIVERSITY MANAGER (Super Admin — full read/write access to all modules) └── AMT HEAD (Division head — monitors 4 depts, approves curriculum, reviews reports) ├── POWERPLANT CHIEF INSTRUCTOR (Manages dept · Creates tasks · Submits curriculum) │ ├── Instructors / Teachers (Assigned tasks, manage classes, update students) │ └── Students (Enrolled, tracked by GPA, attendance, at-risk AI flag) ├── AVIONICS CHIEF INSTRUCTOR │ ├── Instructors / Teachers │ └── Students ├── AIRFRAME CHIEF INSTRUCTOR │ ├── Instructors / Teachers │ └── Students └── STRUCTURE CHIEF INSTRUCTOR ├── Instructors / Teachers └── Students DATA FLOW: CI creates task → teacher assigned → reminders fire at configured intervals → task completed → CI notified → AMT auto-report CI submits curriculum update → AMT Head reviews → Approve / Return with notes → CI notified → version archived Deadline passes unmet → 24h escalation push → CI + AMT Head alerted → shown in overdue report Weekly cron → auto-generates PDF report → delivered to AMT Head email at 07:00 Monday
💡 Suggested Feature Improvements

🤖 AI Risk Detection Engine

Predict tasks likely to go overdue using instructor workload history, curriculum complexity, and past deadline reliability. Flag at-risk items 5+ days early so AMT Head can intervene. Shown as red risk scores on each task.

📱 Mobile Push Notifications (FCM)

Firebase Cloud Messaging sends push alerts directly to instructor phones when tasks are assigned, approaching deadlines, or curriculum updates are approved. Works offline — queued and delivered when back online.

📊 Student Progress Analytics

Track per-student attendance, assessment scores, lab hours, and pass/fail predictions. Auto-alert CIs when any student crosses an at-risk threshold before exam season. Displayed on CI student tab.

🔄 Curriculum Versioning System

Every curriculum update creates a diff-tracked version. AMT Head compares current vs. previous side by side before approval. Full audit trail stored — required for ICAO/ECAA accreditation evidence.

⚙️ Automation Features

⏰ Smart Reminder Engine (BullMQ)

Job queues fire reminders at 30d / 15d / 7d / 3d / 1d / same-day / overdue-24h via email + push + in-app. Auto-escalates to AMT Head if task is unaddressed 48h after overdue. Configurable per task.

📋 Weekly Auto-Report (Cron)

Every Monday 07:00, cron generates a PDF summary of all departments' task completions, overdue counts, curriculum progress, and new at-risk students. Emailed to AMT Head and Admin automatically.

🎯 Hoshin Kanri Goal Cascade

ICAO accreditation goals cascade to department-level curriculum milestones, which cascade to individual instructor tasks. Every action is traceable to a strategic objective. Shown on admin strategic goals page.

🏆 Kaizen Performance Scoring

Weekly automated computation of each instructor's and department's score based on on-time completion rate, curriculum adherence, and student pass rates. Discipline Index drives department ranking.
